《Android APP相关的东西》之工具篇：论AS常用快捷键的必要性

发表于 2016-10-23   |   分类于 Android   |     |   阅读次数   |  

论个猫，没什么好论的，一个字：记住并坚持使用！

比如有些变量什么的完全没必要自己去手写是吧?比如要重构什么的没必要鼠标点来点去是吧？比如相要方法的参数提示没必要删掉重写是吧？诸如此类的。

所以,这里给出那些常用的快捷键，自己也算是个总结记录。但是别点完喜欢就完了，使用！使用！一定要使用啊！

功能 快捷键 功能 快捷键
格式化代码 command + alt + L 运行程序到设备 Ctrl + R
导包、修正 alt + enter 运行Debug到设备 Ctrl + D
复制行 command + D 全局搜索文件 Double Shift
本类查找 command + F 单词大小写转换 command + shift + U
全局查找 command + shift + L 包围代码块 command + alt + T
//注释 command + / Generate代码 command + N
代码片段补全 command + J 从VCS更新代码 command + T
提取Field变量 command + alt + F 向VCS提交代码 command + K
提取Local变量 command + alt + V 最近文件列表 command + E
提取方法 command + alt + M 跳转至父类 command + U
撤销 command + Z 行移动代码 alt + shift + up/down
取消撤销 command + shift + Z 块移动代码 command + shift + up/down
列编辑 alt + 光标移动 代码跳转 command + 鼠标单击
代码补全 command + shift + enter 方法参数提示 command + P
查找类 command + O 行移动代码 alt + shift + up/down
重命名 command + F6 查看函数定义 command + Y
返回代码跳转原始位置 command + [ 返回代码跳转目标位置 command + ]
折叠代码 command + “-“ 展开代码 command + “+”
定位到方法名单词首尾 alt + left/right 定位到本行首尾 command + left/right
重构菜单 ctrl + T 跳转代码错误位置 shift+F2
从剪切板选择最近内容复制 command+shift+V AS项目之间切换 command + ~(esc下面那个)
关闭当前文件 command + W 关闭AS command + Q

注：

1.使用的是默认的AS快捷键。

2.由于木有win环境，所以给出的是Mac版的,但是基本上把command换成ctrl就差不多了,个别有差异。

3.由于是想起哪里写哪里的，给出的功能排序有点乱，担待点啊/笑

4.顺便吐槽一下MarkDown的表格语法，好生蛋疼~
